# AI-ML-Football-Analysis-System-
AI-based football analysis system that uses computer vision and ML to analyze match videos. It tracks players and the ball, and generates insights like heatmaps, player speed, and ball possession to support data-driven performance analysis.

## Setup
1. Download model weights: [yolov8x.pt](https://github.com/ultralytics/ultralytics)
2. Place input videos in `input_videos/` folder
3. Run `pip install -r requirements.txt`
